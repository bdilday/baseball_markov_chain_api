Baseball Markov Chain API landing page

This is the landing page for the Baseball Markov Chain API server

You can use the service by making a get request to https://baseball-markov-chain.herokuapp.com/baseball_markov_chain

possible query parameters are

  • p0, p1, p2, p3, p4: probability for an out, single, double, triple, HR
  • seq_length: length of the returned sequence. default is 10, maximum allowed is 100
  • max_score: any runs above the max_score are treated as being max_score. default is 10, maximum allowed is 30
  • nbases: number of bases. default is 3, maximum allowed is 6
  • nouts: number of outs. default is 3, maximum allowed is 10

The request will return a json string which includes:

  • p0-p4: the probabilities applied
  • seq_length: the length of the returned sequence
  • seq: an array of state summaries
  • state_summaries: includes the keys: manN, outN, runsN, with value the probability the state has N (men-on-base, outs, runs scored), respectively.

    Also includes mean_runs, the mean number of runs up to this point in the sequence

Here is an example query to get a sequence of length 10, with the default probabilities

http://baseball-markov-chain.herokuapp.com/baseball_markov_chain?seq_length=10&p0=0.69&p1=0.23&p2=0.05&p3=0.005&p4=0.025&max_score=10
